Installation Guide

  1. Download Extension

    Click the "Download Extension" button at the top of this page.

  2. Save the File

    Choose a location on your computer to save the downloaded file.

  3. Extract the Folder

    Unzip the downloaded file to extract the GitSync folder.

  4. Open Chrome Extensions

    In Google Chrome, navigate to chrome://extensions or click on "Manage Extensions" from the menu.

  5. Enable Developer Mode

    Toggle on the "Developer mode" switch in the top right corner of the extensions page.

  6. Load the Extension

    Click "Load unpacked" and select the extracted GitSync folder.

  7. Verify Installation

    GitSync should now appear in your list of installed extensions.

  8. Pin the Extension

    Click the puzzle piece icon in Chrome's toolbar and pin GitSync for easy access.

  9. GitHub Authentication

    Click the GitSync icon and follow the prompts to authenticate with GitHub (first-time use only).

  10. Grant Permissions

    Review and allow the necessary permissions for GitSync to function properly.

  11. Link or Create Repository

    Choose to link an existing repository or create a new one if needed.

  12. Start Syncing!

    You're all set! Enjoy seamless code synchronization across your platforms.
